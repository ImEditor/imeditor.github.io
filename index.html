Qui sommes-nous ?

ImEditor est un éditeur de photo développé par deux lycéens dans le cadre de la spécialité ISN proposée en terminale. Notre objectif est de proposer un logiciel fonctionnel et utile et de continuer à l'améliorer même au-delà du temps scolaire. Il s'agit donc d'un véritable projet en développement actif.

Comment participer ?

ImEditor est un logicel libre sous licence GNU GPL (v3), qui permet donc la collaboration dans le but d'améliorer le logiciel. Le code source est publié sur Github à cette adresse.

Nous contacter

Pour nous donner votre avis, nous proposer une amélioration ou tous simplement nous laisser un message, vous pouvez nous contacter sur Github
